# Flashcard AI

A Streamlit app that converts PDFs or pasted notes into high-quality flashcards and helps students practice with active recall and lightweight spaced repetition.

## Run locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Set your API key in `.env`:

```env
GROQ_API_KEY=your_api_key_here
```

3. Start the app:

```bash
streamlit run app.py
```

## Note

Do not commit `.env` to version control.
